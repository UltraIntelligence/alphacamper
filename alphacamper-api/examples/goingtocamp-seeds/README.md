# GoingToCamp Seed Directory

Put one JSON file per known white-label domain in this folder when you want to seed broader Canadian coverage before a live catalog connector is ready.

Expected filename pattern:

- `novascotia.goingtocamp.com.json`
- `manitoba.goingtocamp.com.json`
- `parcsnbparks.info.json`
- `nlcamping.ca.json`
- `longpoint.goingtocamp.com.json`
- `algonquinhighlands.goingtocamp.com.json`
- `maitlandvalley.goingtocamp.com.json`
- `saugeen.goingtocamp.com.json`
- `stclair.goingtocamp.com.json`

Each file should follow the same structure as [../goingtocamp-seed.example.json](../goingtocamp-seed.example.json).
